# NextDeck v1.0.0

First full release of NextDeck.

Highlights:
- Multi-page Nextion-to-desktop control deck
- App importing with automatic metadata and icon capture
- Custom tile names, artwork, and shortcut keys
- Dark and light themes
- 3 x 2 fullscreen-friendly layout option
- About and Settings panels
- Packaged Windows app build in `dist/NextDeck.exe`

Branding and UI:
- NextDeck logo used for the app window and packaged executable icon
- Background artwork integrated into the main shell
- Rounded tile and panel styling
- Cleaner header with settings moved into the Settings panel

Build:
- Run `.\build_app.ps1`
- Output: `dist\NextDeck.exe`
